<h2>🧐 Features</h2>

Some key features of this project:

* Movie Search: Easily search for movies and get detailed information.
* Favorites List: Save movies to a favorites list for quick access later.
* Commenting System: Share your thoughts on movies and read others' comments.
* Responsive Design: Optimized for desktops, tablets, and phones.

<h2>🛠️ Installation Steps:</h2>

<p>1. cd client</p>

```
npm i & npm run dev
```

<p>2. cd server</p>

```
npm i & npm run dev
```
